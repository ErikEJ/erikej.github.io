I’m excited to announce the launch of the **SSMS Extension Gallery**:

👉 [Find it here](https://erikej.github.io/SsmsExtensions/)

This new page gives SQL Server developers and DBAs a quick overview of tools that can improve day-to-day work in SQL Server Management Studio 22.

The goal with this launch is simple: make it easier to discover both free and commercial SSMS extensions in one place.

I got inspired to build this by the blog post [from Brent Ozar here](https://www.brentozar.com/archive/2026/05/announcing-the-ssms-statisticsparser-extension/).

If you build or maintain an SSMS 22 extension and want it included, please [open an issue or pull request](https://github.com/ErikEJ/SsmsExtensions/issues).
